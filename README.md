
# Descriptions

- 긍정/부정 사전(군산대)을 NaiveBayes 분류기에 학습시키고, 문장의 긍정부정을 찾도록 함
- 뉴스를 읽어서 기자별 성향을 찾아내고 자기부정(그땐 맞고 지금은 틀리다, 내로남불 등) 경향을 잡아내기 위한 코드의 일부.
